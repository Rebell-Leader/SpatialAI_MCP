# Archived: hackathon project artifacts

These are **historical** documents from the hackathon that started this project.
They are kept for provenance but are **not current** and should not be used to
understand what the project does today — for that, see
[`README.md`](../../../README.md) and [`AGENTS.md`](../../../AGENTS.md).

Contents:

- `project_details.md` — the original strategic write-up. Describes the early
  **Continue.dev + Docker + execution-tool** vision (pipelines run *by* the MCP
  server). The project has since pivoted to a **provider-neutral** co-pilot whose
  server is **read-only validation/analysis**; those execution tools are roadmap,
  not shipped. Treat tool lists here as aspirational/frozen.
- `PRESENTATION_SCRIPT_4MIN.md` — a hackathon demo pitch script.
- `On-boarding for project ….md` — onboarding lecture notes.
- `SPATIAL_TRANSCRIPTOMICS_ENHANCEMENTS.md` — early enhancement notes from the
  OpenProblems onboarding.

The accurate, maintained domain context now lives in [`context/`](../../../context/)
and the skills in [`skills/`](../../../skills/).
